## Key Implementation Details

### Monkey-Patching Pattern
Tests import controller at module level:
```python
from util.controller import sim
```

Runner monkey-patches the controller before tests run:
```python
from util import controller
controller.sim = hw  # hw = HardwareController instance
```

This allows tests to receive HardwareController instead of SimController without changing test code.

### Usage Examples

```bash
# Run tests against hardware (default port /dev/ttyACM0)
make test-hil

# Run with custom port
PORT=/dev/ttyUSB0 make test-hil

# Flash firmware and run tests
make test-hil FLASH=path/to/firmware.bin

# Command line usage
cd test/integration && python3 run_tests.py --hardware --port /dev/ttyACM0 --flash firmware.bin
```
