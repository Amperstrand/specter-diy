## Method Signatures:

### start() -> None
- Verifies ST-Link connection
- Resets device
- Waits for boot via UART

### load() -> None  
- Opens UART channel
- Drains buffered data
- Verifies device responsive via TEST_STATUS
- Unlocks with TEST_PIN

### shutdown() -> None
- Closes UART channel
- Resets state tracking

### query(data, commands=[]) -> bytes
```python

    commands: list[Union[str, bytes, bool, int, None]] = []

```
- Matches SimController.query() signature
- Returns bytes format expected by tests
- Includes timeout handling with soft reset + retry

### reset(hard=False) -> None
```python
def reset(self, hard: bool = False) -> None
```
- Soft reset: TEST_RESET UART command (default)
- Hard reset: STLinkManager.reset() when hard=True

## Return Format:

- Error responses: b"error: <message>"
